# 🤖 Agentic AI Research Assistant

## 📌 Overview
This project is an AI-powered research assistant that uses Generative AI and agentic workflows to process user queries.

## 🚀 Features
- Prompt Engineering
- LLM Integration (OpenAI)
- Agent-based decision making
- Data analysis using Pandas
- Logging system
- Streamlit UI

## 🧠 Tech Stack
- Python
- OpenAI API
- Pandas
- Streamlit

## ⚙️ Setup
1. Clone repo
2. Install dependencies:
   pip install -r requirements.txt
3. Add API key in `.env`
4. Run:
   python main.py
   OR
   streamlit run app.py

## 🎯 Use Cases
- AI-powered Q&A
- Data-driven insights
- Research automation
